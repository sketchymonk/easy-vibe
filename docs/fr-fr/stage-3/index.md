## À qui s'adresse cette étape

- Développeurs avancés ayant une expérience en développement full-stack et souhaitant relever le défi d'applications plus complexes
- Ingénieurs souhaitant maîtriser les technologies de développement multiplateforme
- Explorateurs souhaitant approfondir leur compréhension du développement d'applications natives IA
- Blogueurs techniques souhaitant établir leur marque technique personnelle

## Prérequis

- Avoir complété l'étape « Développement junior à intermédiaire », ou avoir une expérience en développement full-stack
- Être familier avec les frameworks frontend (tels que React/Vue) et le développement backend
- Comprendre les concepts de base de l'IA et l'utilisation des API
